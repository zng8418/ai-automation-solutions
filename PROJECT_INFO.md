## 部署说明

### GitHub Pages 部署
1. 创建GitHub仓库
2. 上传所有文件
3. 设置 → Pages → Source: main / (root)
4. 访问: https://[username].github.io/[repo-name]

### 自定义域名（可选）
1. 添加CNAME文件
2. 配置DNS解析
3. 启用HTTPS
